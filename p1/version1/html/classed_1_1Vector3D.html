double ed::Vector3D::angulo ( ed::Vector3D const &  v) const

Observador del angulo del vector, que posee la siguiente formula:
sqrt(v1*v1+v2*v2+v3*v3)

Note
Observador declarado como const
Parameters
ed::Vector3Dconst &v → Referencia constante de tipo Vector3D
que posee el valor de cada una de las componentes del vector.
perteneciente al espacio de nombres ed
Precondition
*
Note
*Se podría definir como complemento a la postcondición la
la estructura de control if(angulo < -1) && if(angulo > 1) ya que
el valor del angulo debe de estar entre [-1,1]. En caso de que supere
por alguno de los dos extremos, se asigna el extremo.
Postcondition
Se realiza la diferencia entre valorDevuelto (angulo) y la
operación acos(dotProduct(v)/(modulo())*(v.modulo())). Si el
resultado en valor absoluto es menor que la COTA_ERROR, se tomará
como verdadero la condición del aserto.
Returns
Devuelve el angulo que forma el vector en cuestión con el que se
le ha pasado por Referencia. De tipo double (Real).
